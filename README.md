# Spellchecker

Console app that check spelling of given text. App is using hashtable data structure which makes spellcheck very fast.

## Description

Given the app architecture implement the HashTable and WordChecker class.

[Full description of the task](https://www.ics.uci.edu/~goodrich/teach/ics23/LabManual/LostForWords/)
